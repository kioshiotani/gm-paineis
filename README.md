# gm-paineis

Site desenvolvido durante a disciplina **Introdução ao Desenvolvimento Web (SCC0219)** no primeiro semestre de 2026.

## Features a serem implementadas

### Gestão de Serviços
* **Agenda de serviços:** Painel exclusivo para o administrador gerenciar compromissos.
    * A agenda também exibirá a disponibilidade do prestador de serviços.
* **Solicitação de serviço:** Formulário para coleta de dados essenciais:
    * Nome completo
    * Telefone/E-mail
    * Endereço
    * Data e horário desejado
* **Notificação ao prestador:** Quando um cliente solicitar um serviço enviar um email com as informações do cliente ao prestador de serviços.

### Interatividade e Conteúdo
* **Espaço para avaliação:** Sistema para feedback sobre o prestador de serviço.
* **Seção informativa:** Conteúdo sobre a importância e utilidade da limpeza de painéis solares.
* **Contato direto:** Espaço dedicado para exibir os meios de contato do prestador de serviços.
